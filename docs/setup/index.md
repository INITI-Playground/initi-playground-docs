# Initial Setup, Calibration, and User Guide

**Welcome to the INITI Playground setup and user guide.**

This guide is intended for installers and operators setting up INITI Playground for the first time or preparing the system for daily operation.

In this section, you will learn how to complete the initial setup, calibrate the sensor, connect the system to the internet, install updates, and configure essential settings. Following these guides will ensure your INITI Playground is properly prepared and ready for use.
