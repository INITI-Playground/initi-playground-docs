# Connecting to the Internet

INITI Playground supports automatic updates and online game downloads.  
To receive update notifications and install new content, the system must be connected to the internet via Wi-Fi.

---

## Open Wi-Fi Settings

1. Open the UI and navigate to the `Settings` tab at the top of the screen.
2. In the `General` settings section, locate the **Wi-Fi** settings shown below.

<div class="center imageWidth" markdown>
![Wi-Fi settings](assets/wifi-settings.jpg "Wi-Fi settings")
</div>

---

## Connect to a Wi-Fi Network

- In the **NETWORK / WIFI** dropdown list, select your Wi-Fi network.

<div class="center imageWidth" markdown>
![Wifi settings opened](assets/wifi-settings-opened.jpg "Wifi settings opened")
</div>

- Enter the Wi-Fi password.

<div class="center imageWidth" markdown>
![Wifi password](assets/wifi-settings-password.jpg "Wifi password")
</div>

- Click the `CONNECT` button.

If the connection is successful, a notification will appear:

**"Successfully connected to network XXX."**

If the connection fails (for example due to an incorrect password), an error message will appear:

**"Unable to join the network."**

<div class="center imageWidth" markdown>

=== "Successfully connected"

    ![Connected to Wifi](assets/wifi-settings-success.jpg "Connected to Wifi")

=== "Connection error"

    ![Wifi error](assets/wifi-settings-error.jpg "Wifi error")

</div>

!!! info
    An internet connection is required to receive system updates and download new games from the library.

---

## Disconnect from a Wi-Fi Network

To disconnect from the current network (for example when switching networks), click the `DISCONNECT` button.  
A confirmation notification will appear once the system is disconnected.

<div class="center imageWidth" markdown>
![Wifi disconnect](assets/wifi-settings-disconnect.jpg "Wifi disconnect")
</div>
