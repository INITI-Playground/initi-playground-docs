# How to restart or shut down INITI Playground

There are multiple ways to safely restart or shut down the INITI Playground system.

## Using the on-screen shutdown button

- Locate and click the shutdown icon in the top-right corner of the screen, as shown below.

<div class="center imageWidth" markdown>
![Shutdown button](assets/shutdown-button.jpg "Shutdown button")
</div>

- A dialog box will appear with available options.
- Select `Restart`, `Shut Down`, or `Cancel` by clicking the corresponding button.

<div class="center imageWidth" markdown>
![Shutdown options](assets/shutdown-options.png "Shutdown options")
</div>

## Using the shutdown hotkey

INITI Playground also supports a keyboard shutdown shortcut.

- Press and **hold the `ESC` key for 5 seconds**.
- The system will automatically begin the shutdown process.

This feature is especially useful for installations that use external buttons, kiosks, or custom control panels that simulate keyboard input.
