The system supports an unlimited number of players interacting simultaneously in front of the interactive wall.

## Operation and Features

??? custom-faq "Can INITI Playground run automatically without a supervisor?"

    Yes, using the Auto Game Launch feature in the Settings tab, you can configure a designated game to start automatically when the system powers on. This is ideal for public spaces requiring continuous operation.

??? custom-faq "How do I safely shut down INITI Playground?"

    You can shut down using the shutdown icon in the top-right corner of the UI. As an additional option, holding the `ESC` key for 5 seconds will safely power off the system — useful for setups with custom hardware buttons in hotels, museums, or public spaces.

??? custom-faq "How do I enable or disable touch control in games?"

    Touch control can be toggled in the `Games` tab by clicking the touch icon in the top-right corner. Enabling touch allows players to select and launch games directly from the wall. Disabling touch ensures that only the wireless keyboard can control game selection.

## Technical and Hardware Questions

??? custom-faq "Is INITI Playground compatible with LED walls?"

    Yes, the system works seamlessly with LED walls, which are ideal because they are less sensitive to changes in ambient light.

??? custom-faq "Are the projector and sound system included?"

    No, these components are not included. However, we offer advice to help select the most suitable projector and audio setup for your space.

??? custom-faq "How long does the installation take?"

    Mounting the sensor and projector is the most time-consuming part. After that, the rest of the installation and calibration typically takes about 30 minutes.

??? custom-faq "What are the minimum installation requirements?"

    Recommended wall dimensions are at least 3 meters wide and 2 meters high, with a maximum projection area of 16x9 meters. Players should maintain a minimum distance of 3 meters from the wall. Proper spacing ensures accurate gameplay and safe interaction.

??? custom-faq "What instruments can be used to play the games?"

    Most games are designed for soft balls. Additional options include a fishing rod or paint roller, and all games can also be played using hands.

??? custom-faq "Are there any special requirements for soft balls?"

    Foam balls should have a minimum diameter of 7 cm to ensure reliable detection.

??? custom-faq "Can I perform a factory reset if needed?"

    Yes, a factory reset clears all installed games, user settings, and leaderboards, restoring the system to its default state. After a reset, updating the system and games to the latest versions is recommended.

??? custom-faq "How do I update games and the system?"

    Updates can be managed in the Settings tab under the 'About' section. If a new version is available, click `UPDATE` to download and install. Internet connection is required.
